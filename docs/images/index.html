The N.A. Cleantime Calculator (NACC)

About NACC

The N.A. Cleantime Calculator is a DHTML cleantime calculator, designed by N.A. members, for N.A. members. Unlike some calculators that use PHP, this requires that the user's browser support JavaScript. It's pretty easy to install and customize (covered in greater detail below).

This uses the same formula we always use when we calculate cleantime in N.A. meetings, and assign keytags:

  • If the person has 0 or more days, they should have a white key tag.
  • If the person has 30 or more days, they should have an orange tag.
  • If the person has 60 or more days, they should have a green tag.
  • If the person has 90 or more days, they should have a red tag.
  • At this point, we start measuring by months.
  • If the person has 6 or more months, they should have a blue tag.
  • If the person has 9 or more months, they should have a yellow tag.
  • If the person has 12 or more months (one year), they should have a glow-in-the-dark tag.
  • If the person has 18 or more months (a year and a half), they should have a gray tag.
  • At this point, we start measuring by years.
  • If the person has two or more years, they should get a black tag for each year more than one.

The NACC will allow the visitor to enter their cleantime, then displays a natural English phrase that reports their cleantime in total days. If they have more than 90 days, it will also mention how many years, months and days they have.

The NACC is completely JavaScript-based. This means that any Web site that wants to use it, can use it. It does not rely on special server technologies like PHP or Ruby, nor does it require a plugin on the user's browser, like Flash. It was meant to be taken by Web designers of all skill levels, and used in their site.

The NACC is pretty easy to "theme." It uses CSS to do all of its presentation, and low-specificity CSS for as many elements as possible. The code will validate XHTML 1.1, so it will work in almost any page out there. The tags are displayed as PNG files (Note how attractive they are), so they will look good against any background. We use special CSS for Internet Explorer 6 and below, which don't display PNG as easily as do other browsers.

Basically, NACC is a completely "plug and play" cleantime calculator. It's easy to install and use. It's attractive and fun, and you can get fancy with it if you so desire.

The NACC is completely free for anyone to use. It is designed specifically for N.A. members, so others may not find much utility in it.

It is being shared from a deep gratitude for N.A. and all it has done for us.
